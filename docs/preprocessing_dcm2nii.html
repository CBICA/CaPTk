DICOM to NIfTI conversion

This tool converts a DICOM series into the Neuroimaging Informatics Technology Initiative (NIfTI) file format.

REQUIREMENTS:

  1. A folder with the continuous sequence of a DICOM series.

USAGE:

  1. Launch the tool using the 'Preprocessing' -> 'DICOM to NIfTI' menu option.
  2. Specify the directory containing the DICOM images in the series, and the output filename.
  3. Click on "Confirm".
  4. This can also be used from the command line: 
    ${CaPTk_InstallDir}/bin/Utlities.exe -i C:/test/dicomFolderWithSingleSubject -o C:/PathToOutputFolder/result.nii.gz -d2n
